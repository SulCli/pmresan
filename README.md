# pmresan
پیام رسان telebot


این سورس کد هاش از نظر خودم اصلا به درد نمیخوره ولی خب به دلایلی اپن میشه برای استفاده همگانی

پیشنیاز: پایتون3 و ردیس داخل سرور نصب باشد!

اطلاعاتتشو رو داخل فایل 

config.py

بزارید!

برای نصب: ./install.sh
و برای ران کردن سورس اول دستور

tmux

و بعد فایل 

./run.sh

اجرا بشه!
